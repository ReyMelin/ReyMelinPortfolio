About Me

Hi! I'm a passionate web developer and creative technologist with a love for building meaningful digital experiences. My journey in tech started with a curiosity about how things work and evolved into a career focused on creating applications that make a difference.

I specialize in frontend development with React, creating interactive user interfaces, and implementing smooth animations that enhance user experience. I'm particularly interested in accessibility, performance optimization, and building tools that help people express themselves creatively.

When I'm not coding, you'll find me exploring new design trends, contributing to open-source projects, or working on creative side projects like interactive coloring books and emotion journaling apps.
